## Travel App API 
#### Started 

- Install the required dependecies
```
pnpm i
```
- Rename or create .env with requied value

- Start the development server
```
pnpm dev
```
- For Production server 
``` 
pnpm build 
```
and 
```
pnpm start
```
- For database generation and migration 
```
pnpm db:generate
pnpm db:migrate
```
- To view Schema 
```
pnpm db:studio
```
